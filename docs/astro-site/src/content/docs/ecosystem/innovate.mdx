**Innovate** is a Python library for fitting, forecasting, and visualising diffusion models — the mathematical models that describe how new innovations, technologies, or policies spread through a population over time.

Built for **health economics and outcomes research (HEOR)**, it supports Bass, Gompertz, logistic, and custom model specifications with multiple estimation backends including MLE, Bayesian (PyMC), and GPU-accelerated (JAX).

---

## Key Features

<ul class="feature-list">
  <li><strong>Multiple Model Families</strong> — Bass, Gompertz, Logistic, Gumbel, and custom model specifications with flexible parameter constraints.</li>
  <li><strong>Multiple Estimation Methods</strong> — Maximum Likelihood Estimation (MLE), Bayesian inference via PyMC, and non-linear least squares.</li>
  <li><strong>GPU Acceleration</strong> — Optional JAX backend for large-scale simulation and gradient-based optimisation.</li>
  <li><strong>Comprehensive Diagnostics</strong> — Goodness-of-fit statistics, residual analysis, confidence intervals, and model comparison (AIC/BIC).</li>
  <li><strong>Forecasting</strong> — Out-of-sample prediction with uncertainty quantification and ensemble forecasting.</li>
  <li><strong>Visualisation</strong> — Publication-ready plots via Matplotlib: fitted curves, forecasts, diagnostics, and comparative charts.</li>
  <li><strong>Arrow Interchange</strong> — Native support for Apache Arrow and Parquet data formats for seamless interoperability.</li>
  <li><strong>Polyglot API</strong> — The core algorithm is implemented in Rust for performance, with Python bindings via PyO3.</li>
</ul>

## Quick Start

```python
import pandas as pd
from innovate import fit_model, predict_model, summarise_model

# Sample adoption data
data = pd.DataFrame({
    'time': range(1, 13),
    'adoptions': [1, 3, 8, 18, 35, 60, 90, 125, 160, 195, 225, 245]
})

# Fit a Bass diffusion model
result = fit_model(data, model='bass', method='mle',
                   target_col='adoptions', time_col='time')

# View summary
print(summarise_model(result))

# Forecast 8 periods ahead
forecast = predict_model(result, horizon=8)
print(forecast)
```

## Architecture

Innovate's architecture separates the mathematical kernel (implemented in **Rust** for performance and numerical stability) from the user-facing **Python API**. This design enables:

- **Fast computation** of diffusion curve likelihoods and gradients
- **Safe multi-threading** for ensemble and bootstrap methods
- **Polyglot potential** — the Rust kernel can be exposed to other languages in the future
- **Arrow-native data interchange** between the Rust core and Python runtime

## When to Use Innovate

| Use Case | Description |
|----------|-------------|
| **Technology Adoption Forecasting** | Predict how quickly a new health technology or drug will be adopted across markets |
| **Policy Impact Analysis** | Model the diffusion of a policy change through a population or organisation |
| **Market Sizing** | Estimate total addressable market and adoption curves for new products |
| **Epidemiological Modelling** | Adapt diffusion models for infectious disease spread with intervention effects |
| **Innovation Research** | Study the characteristics of innovation diffusion across different sectors |
