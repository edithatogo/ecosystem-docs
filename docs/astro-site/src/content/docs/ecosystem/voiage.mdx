**Voiage** is a cross-domain **Value of Information (VOI)** analysis library that speaks seven programming languages. It provides a consistent, well-tested API for computing Expected Value of Perfect Information (EVPI), Expected Value of Partial Perfect Information (EVPPI), Expected Value of Sample Information (EVSI), and related metrics — whether you work in Python, R, Julia, TypeScript, Go, Rust, or .NET.


---

## Key Features

<ul class="feature-list">
  <li><strong>7 Language Implementations</strong> — Python, R, Julia, TypeScript, Go, Rust, and .NET with a unified API design across all languages.</li>
  <li><strong>EVPI Calculation</strong> — Expected Value of Perfect Information using Monte Carlo simulation with multiple sampling strategies.</li>
  <li><strong>EVPPI by Regression</strong> — Expected Value of Partial Perfect Information via Gaussian Process regression, spline models, and linear models.</li>
  <li><strong>EVSI</strong> — Expected Value of Sample Information using nested Monte Carlo and moment-based methods.</li>
  <li><strong>EVPPI from Regression</strong> — Advanced non-parametric regression methods for EVPPI estimation in high-dimensional parameter spaces.</li>
  <li><strong>Multiple Backends</strong> — NumPy/SciPy, JAX, PyTorch, and pure Python implementations depending on your performance needs.</li>
  <li><strong>Plots & Diagnostics</strong> — Built-in visualisation of VOI curves, expected loss, and population EVPI.</li>
  <li><strong>CLI Tool</strong> — Command-line interface for running VOI analyses without writing code.</li>
</ul>

## Quick Start

```python
import numpy as np
from voiage import evpi, evppi

# Simulate net benefit samples: 1000 samples x 3 treatment strategies
n_samples = 1000
n_strategies = 3
nb = np.random.normal(loc=[100, 120, 110], scale=[30, 35, 32],
                      size=(n_samples, n_strategies))

# Calculate EVPI
evpi_value = evpi(nb)
print(f"EVPI: {evpi_value:.2f}")

# Calculate EVPPI for one parameter of interest
x = np.random.uniform(0, 1, n_samples)
evppi_value = evppi(nb, x, method='gp')
print(f"EVPPI: {evppi_value:.2f}")
```

## VOI Methods Reference

| Method | Description | Implementation |
|--------|-------------|----------------|
| **EVPI** | Expected Value of Perfect Information | Monte Carlo integration over joint parameter distribution |
| **EVPPI** | Expected Value of Partial Perfect Information | Gaussian Process regression, spline, linear models |
| **EVSI** | Expected Value of Sample Information | Nested Monte Carlo, moment-based estimators |
| **EVPPI (regression)** | Non-parametric EVPPI | GAM, GP, and random forest regression methods |
| **Expected Loss** | Opportunity loss under current information | Calculated from net benefit distributions |
| **Population EVPI** | Scaled EVPI for target population | Multiplies per-patient EVPI by population size |
