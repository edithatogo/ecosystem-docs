**Lifecourse** is a Python microsimulation engine for modelling individual patient trajectories across a full lifetime horizon. It supports multiple simulation paradigms — **state-transition (Markov) models**, **discrete-event simulation (DES)**, and **agent-based modelling (ABM)** — all within a unified, modular framework.

Built for **health economics and outcomes research**, Lifecourse enables analysts to simulate complex disease progression, treatment pathways, and health outcomes at the individual level, with built-in support for costs, utilities, and uncertainty analysis.

## Quick Start

```python
from lifecourse import MicrosimulationModel, HealthState, Transition

# Define health states
healthy = HealthState("Healthy", cost=0, utility=1.0)
sick = HealthState("Sick", cost=5000, utility=0.7)
dead = HealthState("Dead", cost=0, utility=0.0, absorbing=True)

# Define transitions
healthy.add_transition(Transition(to=sick, rate=0.05))
healthy.add_transition(Transition(to=dead, rate=0.01))
sick.add_transition(Transition(to=dead, rate=0.08))

# Create and run model
model = MicrosimulationModel(
    states=[healthy, sick, dead],
    n_patients=10000,
    time_horizon=40,  # years
    cycle_length=1     # annual cycles
)

results = model.run()
print(f"Mean total cost: ${results.mean_cost():.2f}")
print(f"Mean QALYs: {results.mean_qalys():.2f}")
print(f"ICER: ${results.icer():.2f}/QALY")
```

## Simulation Paradigms

| Paradigm | Best For | Lifecourse Module |
|----------|----------|-------------------|
| **State-Transition (Markov)** | Chronic disease with defined health states and fixed cycle lengths | `lifecourse.markov` |
| **Microsimulation** | Individual patient histories with time-varying covariates | `lifecourse.microsim` |
| **Discrete-Event Simulation (DES)** | Resource-constrained systems with queueing and events | `lifecourse.des` |
| **Agent-Based Modelling (ABM)** | Interactions between heterogeneous agents | `lifecourse.abm` |

---

## When to Use Lifecourse

| Use Case | Description |
|----------|-------------|
| **Cost-Effectiveness Analysis** | Full incremental cost-effectiveness analysis with PSA and CEACs |
| **Disease Progression Modelling** | Simulate natural history of chronic diseases (CVD, diabetes, cancer) |
| **Treatment Pathway Analysis** | Compare sequences of treatments over a patient's lifetime |
| **Health Technology Assessment** | Generate evidence for HTA submissions (NICE, CADTH, PBAC) |
| **Population Health Forecasting** | Project disease burden and healthcare resource utilisation |
| **Value of Implementation** | Assess the value of improving implementation of existing interventions |
