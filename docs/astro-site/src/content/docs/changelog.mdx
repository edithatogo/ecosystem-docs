# Changelog

All notable changes to this central ecosystem documentation site will be documented here.

## [0.1.0] — 2025-05-17

### Added

- **Ecosystem landing page** — Splash page with hero section, project cards, quick links, ecosystem principles, and technology stack overview
- **Project detail pages** — Full content pages for Innovate, Voiage, Lifecourse, Mars, and Starlight Polyglot with key features, quick start examples, external resource links, language support tables, use case guides, and cross-project references
- **Getting Started guide** — Cross-project workflow tutorial showing how to combine all ecosystem tools
- **Contributing guide** — Comprehensive guide covering all ways to contribute across the ecosystem
- **Custom styling** — Ecosystem-branded theme with accent colours, card animations, feature lists, and link cards
- **GitHub Actions deployment** — CI/CD pipeline for automated build and deployment to GitHub Pages
- **Starlight configuration** — Full sidebar navigation, social links, edit links, links validation, and llms.txt support
